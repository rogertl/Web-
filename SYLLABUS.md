# AI 时代现代 React + Next.js 全栈代码理解清单

> 从读懂 AI 代码出发，7 个阶段掌握现代 React + Next.js 全栈开发核心。

---

## 设计原则

1. **由宏观到微观、由基础到高级**，严格遵循认知发展顺序
2. **每个阶段明确前置依赖**，必须先懂前面才能有效学习后面
3. **重要概念螺旋式递进**，第一次浅理解，后续再加深
4. **控制单阶段概念量**，避免一次性塞太多新名词
5. 打分基于 **"读懂 AI 代码"** 的实际需要

## 学习建议

- **每周节奏**：每周重点攻克 1 个阶段，不要一次性学完
- **学习方法**：每阶段结束，让 AI 生成一个对应的小功能代码，用自己的话讲解"这个代码在干什么？数据从哪里来？到哪里去？"
- **验证标准**：能看懂 AI 生成的中型项目代码（包含 Server Action + 数据库 + 权限），就算基本达标
- **螺旋强化**：后面阶段遇到前面概念时，要回过头加深理解
